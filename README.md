# Welcome to My Ngram

## Task
Write a program my_ngram; It will count the number of occurrences per character.

## Installation

*not installable*

## Usage
 
 ```
 ./my_ngram.exe "ARGUMENTS"
 ```
Example output:

```
A: 1
E: 1
G: 1
M: 1
N: 1
R: 1
S: 1
T: 1
U: 1
```
